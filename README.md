# Travel Explorer

Travel Explorer is a responsive web project built using HTML, CSS, Bootstrap, and Node.js. It showcases popular North American destinations such as Toronto, Montreal, and New York through a clean and modern interface.

## Features
• Responsive Bootstrap navigation bar  
• Dropdown menu for organized sections  
• Grid based travel destination layout  
• Image thumbnails and styled sections  
• Fixed footer  
• Node.js server to display Hello World and host the webpage
